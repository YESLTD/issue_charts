= Issue Charts plug-in

This plug-in provides great functionality for graphing issues with Chartkick (http://chartkick.com/)

== Installation

* Clone into your plugins folder: <b>git clone https://github.com/masweetman/issue_charts.git</b>
* Run <b>rake redmine:plugins:migrate RAILS_ENV=production</b>
* Run <b>bundle install</b>
* Restart Redmine
